# skill: basic

## Description

Basic skill template for creating simple informational skills.

## When to Use

- Documenting team processes
- Storing reference information
- Quick answers to common questions
- Guidelines and policies

## Template

```markdown
# skill: skill-name

## Description
Brief description of what this skill covers.

## When to Use
Situations when this skill should be invoked.

## Instructions
Step-by-step guidance or detailed information.

## Examples
Code snippets, examples, or usage patterns.

## Related
Links to related skills or resources.
```

## Creating from Template

```bash
cp skills/templates/basic/README.md skills/your-category/your-skill.md
```

## Tips

- Keep descriptions concise (1-2 sentences)
- Use clear section headers
- Include practical examples
- Link to related skills
